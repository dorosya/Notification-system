Отправка сообщения на сервер

Введите сообщение и отправьте его на localhost:8080/event

Отправка на endpoint: POST http://localhost:8080/event

Параметр: message с введенным значением

Примечание: Для работы этой страницы у вас должен быть запущен сервер на localhost:8080 с обработкой POST-запросов на /event.

Вы можете использовать Node.js с Express, Python Flask или любой другой серверный фреймворк.
